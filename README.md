# RESTful
RESTful for yura.store

To start, you need XAMPP 8.1.2-0

run XAMPP
```
sudo /opt/lampp/manager-linux-x64.run
```


Clone repository
```
sudo chmod -R 777 /opt/lampp/htdocs
cd /opt/lampp/htdocs
git clone https://github.com/yuraMovsesyan/RESTful
```

Open link
```
http://localhost/RESTful/public
```
